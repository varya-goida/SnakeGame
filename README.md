# Snake
## Описание 
Простейшая компьютерная игра "Змейка"  
## Цель проекта
Разработка игры "Змейка", с помощью библеотеке Tkinter 
## Архитектура проекта
├── src/   
│   └── Snake.py
├── Dockerfile
├── README.md
├── .gitignore
├── requirements.txt
## Локальный запуск
# Установка зависимостей
pip install -r requirements.txt
## Примеры использования
За неимением интернета и иных видов досуга запускаете Змейку и, выбрав сложность увлекаетесь занимательным геймплеем собирания точек
